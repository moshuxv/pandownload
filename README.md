# 百度网盘字幕自动导出 + PDF批量下载 — 安装说明

## 插件位置
```
D:\download\baidu-pan-subtitle-extension\
```

## 安装步骤

1. 打开 Chrome，地址栏输入 `chrome://extensions/`
2. 右上角打开 **"开发者模式"** 开关
3. 点击 **"加载已解压的扩展程序"**
4. 选择文件夹 `D:\download\baidu-pan-subtitle-extension`
5. 安装完成，工具栏会出现 📝 图标

## 使用方法

### 1. 扫描目录
进入百度网盘目录页，右侧弹出控制面板，点击 **「🔍 扫描目录」** 按钮自动递归扫描整个目录树。

### 2. 导出视频字幕
扫描完成后，点击 **「🎬 导出视频字幕」** → 自动逐视频打开播放页 → 提取文稿文本 → 下载 .txt → 关闭标签页。最多5个并发。

### 3. 下载PDF文档
扫描完成后，点击 **「📕 下载PDF文档」** → 自动通过百度网盘API获取下载直链 → 批量下载，保持原文件夹结构。最多3个并发。

> 💡 两个操作完全独立，可先后进行，互不影响。如目录下无PDF则"下载PDF文档"不显示文件数。

## 目录结构保持
- PDF 下载后保持百度网盘中的原目录层级
- 例如：`根目录/子目录A/文档.pdf` → 下载为 `子目录A/文档.pdf`
- 特殊字符会被自动替换为下划线

## 注意事项

- 字幕导出：不依赖"导出"按钮，直接从 DOM 提取文稿文本
- PDF 下载：通过百度网盘 API 获取直链，3个并发下载
- 文件名中的特殊字符会被安全替换
- 如需暂停：点击工具栏插件图标 → 取消对应开关
- 浮动状态面板显示在页面右上角，实时显示进度

## 文件结构

| 文件 | 作用 |
|------|------|
| manifest.json | 插件配置 |
| content.js | 注入到视频页的字幕自动提取 |
| content_dir.js | 注入到目录页：递归扫描 + 字幕批量导出 + PDF批量下载 |
| background.js | 后台处理下载和标签页管理 |
| popup.html/js | 开关控制界面 |
| icons/ | 插件图标 |
